# 💼 簡易経営判断ツール（中小企業向け）

**最小限の入力で、自社の経営状況を数値で把握**

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://cashflow-risk-analyzer-zityxm7xbuivhjmwb5ngvv.streamlit.app/)

---

## 📌 このツールについて

中小企業経営者が**自社の経営状況を客観的に把握し、改善策を検討する**ための診断ツールです。

- 💡 **簡単**: 8項目の入力だけで診断完了（5分で完了）
- 📊 **わかりやすい**: 経営者向けの読みやすいレポート
- 🎯 **実用的**: 損益分岐点・資金繰りリスク・必要資金を一度に診断

**銀行融資用の資料ではありません。** 経営者自身が現状を把握し、社内で改善策を検討するためのツールです。

---

## ✨ 主な機能

### 📈 1. 損益分岐点分析
- **月次損益分岐点売上高**を自動計算
- 現在の売上との比較（達成率）
- **営業日別の売上目標**（20/21/22日パターン）

→ 営業チームへの目標展開が可能

### 💰 2. 資金繰りリスク診断
- モンテカルロシミュレーション（10,000回）による確率的分析
- 資金ショート確率を算出
- 12ヶ月先までのリスク予測

→ 「資金繰りが厳しくなる確率」を数値で把握

### 🔥 3. ストレステスト
過去の経済危機レベルのショックが発生した場合の影響を診断：
- リーマンショック（2008年）
- 東日本大震災（2011年）
- コロナショック（2020年）
- ウクライナ戦争（2022年）

→ 最悪のシナリオでのリスクを事前把握

### 💵 4. 必要運転資金の算出
目標リスク水準別（安全/標準/最低限）に必要な運転資金額を逆算：
- 必要運転資金（年間）
- 返済計画（5年返済の場合）
- 既存借入を考慮したCF余剰分析

→ 資金調達の必要性を客観的に判断

### 📄 5. レポート出力
- **Word形式**: 社内検討用の診断レポート
- **Excel形式**: 詳細データ（グラフ・数値）

## 🚀 使い方（3ステップ）

### ステップ1: アクセス
ブラウザで以下のURLを開く（インストール不要）

👉 **https://cashflow-risk-analyzer-zityxm7xbuivhjmwb5ngvv.streamlit.app/**

### ステップ2: 基本情報を入力（8項目・5分）
- 月次売上高
- 現金残高
- 売上原価率
- 月次固定費
- 売上変動率
- 売掛サイト
- 支払サイト
- 在庫回転期間

**業種・従業員数を選ぶと、標準値が自動入力されます**

### ステップ3: レポートをダウンロード
- **Word**: 診断レポート（社内検討用）
- **Excel**: 詳細データ（数値・グラフ）

## 📊 出力される診断項目

### 1️⃣ 損益分岐点分析
```
月次損益分岐点売上高: 1,200万円
現在の売上: 1,500万円
達成率: 125.0% （良好）

営業日別目標:
・20営業日/月: 60.0万円/日
・21営業日/月: 57.1万円/日
・22営業日/月: 54.5万円/日
```

### 2️⃣ 資金ショート確率
```
通常シナリオ: 12%
リーマンショック級: 45%
コロナショック級: 38%
```

### 3️⃣ 必要運転資金（年間）
```
安全水準（5%以下）: 800万円
標準水準（10%以下）: 500万円
最低限（15%以下）: 300万円
```

## 💡 このツールでできないこと

❌ **銀行融資の申込書作成**  
→ 銀行は決算書類とヒアリングが必要。このレポートだけで融資判断はしません。

❌ **精密な財務分析**  
→ あくまで簡易診断。詳細な分析は税理士・会計士にご相談ください。

❌ **個別企業の投資判断**  
→ 投資・M&Aの判断材料としては使用できません。

## 🎓 開発の背景

このツールは、20年以上の金融実務経験（信用金庫・企業評価機関）を持つ開発者が、**中小企業経営者の実務課題**を解決するために開発しました。

多くの中小企業では：
- 資金繰り表を作成していない
- 損益分岐点を把握していない
- リスクを「感覚」で判断している

このツールは、**最小限の入力で、経営判断に必要な数値を提供する**ことを目指しています。
