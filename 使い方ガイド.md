## 🚀 使い方（3ステップ）

### ステップ1: ファイルを配置する

1. ダウンロードした `.ipynb` ファイルを**わかりやすい場所**に移動
   - 例: `C:\Users\TSR\Documents\資金繰りツール\`
   - デスクトップでもOK

### ステップ2: Jupyter Notebookで開く

1. **Jupyter Notebook**を起動（スタートメニュー → Jupyter Notebook）
2. ブラウザでファイル一覧が開く
3. ファイルを保存した場所に移動
4. `phase1_資金繰りシミュレーター_日本語版.ipynb` をクリック

### ステップ3: 実行する

1. 上部メニュー **「Kernel」** をクリック
2. **「Restart & Run All」** をクリック
3. 確認ダイアログで **「Restart and Run All Cells」** をクリック
4. 数秒〜10秒待つ

✅ グラフが2枚表示されれば成功！

## 🎯 自社データで試す方法

### パラメータの変更（セル3を編集）

ノートブック内の **「セル3」** を探してください。
以下のような部分があります：

```python
params = CashFlowParameters(
    monthly_sales=1000,           # ← ここを変更
    cash_balance=800,             # ← ここを変更
    monthly_fixed_cost=320,       # ← ここを変更
    cost_rate=0.60,               # ← ここを変更
    sales_volatility=0.15,
    accounts_receivable_days=30,
)
```

**変更例（飲食店の場合）:**

```python
params = CashFlowParameters(
    monthly_sales=800,            # 月次売上800万円
    cash_balance=300,             # 現金残高300万円
    monthly_fixed_cost=500,       # 固定費500万円
    cost_rate=0.65,               # 原価率65%（飲食は高め）
    sales_volatility=0.20,        # 変動率20%（季節変動あり）
    accounts_receivable_days=0,   # 現金商売
)
```

変更後、もう一度 **「Kernel」→「Restart & Run All」** で実行してください。

## 📧 グラフの活用方法

### PowerPoint・Wordに貼り付け

1. PNGファイルをドラッグ&ドロップするだけ
2. 高解像度なので拡大してもキレイ

### メールで送信

1. PNGファイルを **右クリック**
2. **「送る」** → **「メール受信者」**
3. メールソフトが開いて添付された状態になります

### 印刷

1. PNGファイルを開く
2. **右クリック** → **「印刷」**
3. A4横で印刷するとちょうど良いサイズ

## ❓ トラブルシューティング

### Q: ノートブックが開かない
**A:** Anacondaが正しくインストールされているか確認してください。

### Q: エラーが出る
**A:** エラーメッセージ（赤い文字）をコピーして、Claudeに質問してください。

### Q: グラフが表示されない
**A:** 「Kernel」→「Restart & Run All」をもう一度実行してください。

### Q: PNGファイルが見つからない
**A:** ノートブックを保存した場所と同じフォルダを確認してください。

### Q: 文字が小さくて見づらい
**A:** PNGファイルをダブルクリックして拡大表示してください。高解像度で保存されています。

## 📝 業種別パラメータの目安

### 製造業
- 原価率: 60-70%
- 変動率: 10-15%
- 売掛サイト: 30-60日

### 飲食業
- 原価率: 65-75%
- 変動率: 20-30%（季節変動大）
- 売掛サイト: 0日（現金商売）

### 小売業
- 原価率: 70-80%
- 変動率: 15-20%
- 売掛サイト: 0-30日

### サービス業
- 原価率: 30-50%
- 変動率: 10-15%
- 売掛サイト: 30-45日

### 建設業
- 原価率: 80-85%
- 変動率: 25-35%（プロジェクト型）
- 売掛サイト: 60-90日
